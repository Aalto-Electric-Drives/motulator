helpers.abc2complex(u)¶

Transform three-phase quantities to a complex space vector.

Parameters:

u (array_like, shape (3,)) – Phase quantities.

Returns:

Complex space vector (peak-value scaling).

Return type:

complex

Examples

>>> y = abc2complex([1, 2, 3])
>>> y
(-1-0.5773502691896258j)

helpers.ref_ramp(mdl, w_max=2 * np.pi * 50, tau_max=14.6, t_max=4)¶

Generate an example ramp profile for the speed reference. The load torque changes stepwise.

Parameters:

  • mdl (object) – Model.

  • w_max (float, optional) – Maximum speed in the profile. The default is 2*pi*50.

  • tau_max (float, optional) – Maximum load torque in the profile. The default is 14.6.

  • t_max (float, optional) – Length of the profile. The default is 4.

helpers.ref_step(mdl, w_max=0.8 * 2 * np.pi * 50, tau_max=14.6, t_max=1.5)¶

Generate an example stepwise profile for the speed reference and load torque.

Parameters:

  • mdl (object) – Model.

  • w_max (float, optional) – Maximum speed in the profile. The default is .8*2*pi*50.

  • tau_max (float, optional) – Maximum load torque in the profile. The default is 14.6.

  • t_max (float, optional) – Length of the profile. The default is 1.5.
