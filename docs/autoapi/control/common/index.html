control.common.duty_ratios(u, u_dc)[source]¶

Compute the duty ratios for three-phase PWM.

This compuates the duty ratios for three-phase PWM using a symmetrical suboscillation method. This method is identical to the standard space-vector PWM.

Parameters:

  • u (complex) – Voltage space vector (typically reference value).

  • u_dc (float) – DC-bus voltage (typically measured value).

Returns:

d_abc – Duty ratios (typically reference values).

Return type:

ndarray, shape (3,)
