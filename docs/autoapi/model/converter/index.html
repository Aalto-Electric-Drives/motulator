model.converter¶

This module contains power converter models.

The inverter with the constant DC-bus voltage and the frequency converter with the diode front-end rectifier are modeled.

class model.converter.FrequencyConverter(C=0.000235, L=0.002, u_g=np.sqrt(2 / 3) * 400, w_g=2 * np.pi * 50)[source]¶

Frequency converter.

This models a strong grid, a three-phase diode-bridge rectifier, an LC filter, and a three-phase inverter.

Parameters:

  • C (float, optional) – DC-bus capacitance. The default is 235e-6.

  • L (float, optional) – DC-bus inductance. The default is 2e-3.

  • u_g (float, optional) – Grid voltage (line-neutral, peak). The default is sqrt(2/3)*400.

  • w_g (float, optional) – Grid angular frequency. The default is 2*pi*50.
