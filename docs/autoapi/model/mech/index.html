Mechanics

This class represents the mechanical subsystem.
class model.mech.Mechanics(J=0.015, B=0, tau_L_ext=Step(0.8, 14.6))¶

This class represents the mechanical subsystem.

Parameters:

  • J (float, optional) – Total moment of inertia. The default is .015.

  • B (float, optional) – Viscous damping coefficient. The default is 0.

  • tau_L_ext (function, optional) – External load torque as a function of time, tau_L_ext(t). The default is Step(.8, 14.6).
