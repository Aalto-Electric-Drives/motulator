model.interfaces.solve(mdl, d_abc, t_span, max_step=np.inf)¶

Solve the continuous-time model over t_span.

Parameters:

  • mdl (object) – Model to be simulated.

  • d_abc (array_like of floats, shape (3,)) – Duty ratio references in the interval [0, 1].

  • t_span (2-tuple of floats) – Interval of integration (t0, tf). The solver starts with t=t0 and integrates until it reaches t=tf.

  • max_step (float, optional) – Max step size of the solver. The default is inf.

class model.interfaces.PWM(enabled=True, N=2 ** 12)¶

This class implements carrier comparison of three-phase PWM. The switching instants and the switching states are explicitly and exactly computed from the duty ratios. The switching instants can be used in the ODE solver.

Parameters:

  • N (int, optional) – Amount of PWM quantization levels. The default is 2**12.

  • enabled (Boolean, optional) – PMW enabled. The default is True.
