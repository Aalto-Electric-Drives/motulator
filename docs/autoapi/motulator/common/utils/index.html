class motulator.common.utils.SequenceGenerator(times, values, periodic=False)[source]#

Sequence generator.

The time array must be increasing. The output values are interpolated between the data points.

Parameters:

  • times (ndarray) – Time values.

  • values (ndarray) – Output values.

  • periodic (bool, optional) – Enables periodicity, defaults to False.

Transform three-phase quantities to a complex space vector.

Returns:

Complex space vector (peak-value scaling).

Return type:

complex

Examples

>>> from motulator.common.utils import abc2complex
>>> y = abc2complex([1, 2, 3])
>>> y
