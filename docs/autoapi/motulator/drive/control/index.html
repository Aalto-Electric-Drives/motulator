SpeedController

2DOF PI speed controller.

Package Contents#

class motulator.drive.control.SpeedController(J, alpha_s, alpha_i=None, tau_M_max=inf)[source]#

Bases: motulator.common.control._controllers.PIController

2DOF PI speed controller.

This is an interface for a speed controller. The gains are initialized based on the desired closed-loop bandwidth and the rotor inertia estimate.

Parameters:

  • J (float) – Total inertia of the rotor (kgm²).

  • alpha_s (float) – Reference-tracking bandwidth (rad/s).

  • alpha_i (float, optional) – Integral action bandwidth (rad/s), defaults to alpha_s.

  • tau_M_max (float, optional) – Maximum motor torque (Nm), defaults to inf.
