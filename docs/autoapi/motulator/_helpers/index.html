class motulator._helpers.BaseValues[source]#

Base values.

Base values are computed from the nominal values and the number of pole pairs. They can be used, e.g., for scaling the plotted waveforms.

Parameters:

  • U_nom (float) – Voltage (V, rms, line-line).

  • I_nom (float) – Current (A, rms).

  • f_nom (float) – Frequency (Hz).

  • tau_nom (float) – Torque (Nm).

  • P_nom (float) – Power (W).

  • n_p (int) – Number of pole pairs.

class motulator._helpers.Sequence(times, values, periodic=False)[source]#

Sequence generator.

The time array must be increasing. The output values are interpolated between the data points.

Parameters:

  • times (ndarray) – Time values.

  • values (ndarray) – Output values.

  • periodic (bool, optional) – Enables periodicity. The default is False.
