class motulator.control.common.PWM(pars)[source]

Duty ratio references and realized voltage for three-phase PWM.

This contains the computation of the duty ratio references and the realized voltage. The digital delay effects are taken into account in the realized voltage.

Parameters

pars (data object) – Control parameters.

static six_step_overmodulation(u_s_ref, u_dc)[source]

Overmodulation up to six-step operation.

This method modifies the angle of the voltage reference vector in the overmodulation region such that the six-step operation is reached [1].

Parameters

u_s_ref (complex) – Voltage reference in stator coordinates.

Returns

u_s_ref_mod – Voltage reference in stator coordinates.

Return type

complex

References

1

Bolognani, Zigliotto, “Novel digital continuous control of SVM inverters in the overmodulation range,” IEEE Trans. Ind. Appl., 1997, https://doi.org/10.1109/28.568019

static duty_ratios(u_s_ref, u_dc)[source]

Compute the duty ratios for three-phase PWM.

This computes the duty ratios using a symmetrical suboscillation method. This method is identical to the standard space-vector PWM.

Parameters

  • u_s_ref (complex) – Voltage reference in stator coordinates.

  • u_dc (float) – DC-bus voltage.

Returns

d_abc_ref – Duty ratio references.

Return type

ndarray, shape (3,)
