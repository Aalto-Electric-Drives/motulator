class motulator.model.mech.Mechanics(J=0.015, tau_L_w=lambda w_M: ..., tau_L_t=lambda t: ...)[source]

Mechanics subsystem.

This models an equation of motion for stiff mechanics.

Parameters

  • J (float) – Total moment of inertia.

  • tau_L_w (function) – Load torque as function of speed, tau_L_w(w_M). For example, tau_L_w = b*w_M, where b is the viscous friction coefficient.

  • tau_L_t (function) – Load torque as a function of time, tau_L_t(t).

class motulator.model.mech.MechanicsTwoMass(J_M=0.005, J_L=0.005, K_S=700.0, C_S=0.13, tau_L_w=lambda w_M: ..., tau_L_t=lambda t: ...)[source]

Bases: Mechanics

Two-mass mechanics subsystem.

This models an equation of motion for two-mass mechanics.

Parameters

  • J_M (float) – Moment of inertia of the motor.

  • J_L (float) – Moment of inertia of the load.

  • K_S (float) – Torsional stiffness of the shaft.

  • C_S (float) – Torsional damping of the shaft.

  • tau_L_w (function) – Load torque as function of the load speed, tau_L_w(w_L). For example, tau_L_w = b*w_L, where b is the viscous friction coefficient.

  • tau_L_t (function) – Load torque as a function of time, tau_L_t(t).
