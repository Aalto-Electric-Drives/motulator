Grid-Forming Control
--------------------

These examples demonstrate grid-forming control for grid-connected converters. 
The examples :doc:`/auto_examples/grid_forming/plot_power_synchronization_control_grid_converter_12.5kVA`
and :doc:`/auto_examples/grid_forming/plot_reference_feedforward_power_synchronization_control_grid_converter_12.5kVA`
use a power synchronization loop for synchronizing with the grid [#Har2019]_,[#Har2020]_.
In :doc:`/auto_examples/grid_forming/plot_observer_gfm_control_grid_converter_12.5kVA`
a disturbance observer-based control structure is used [#Nur2024]_.
